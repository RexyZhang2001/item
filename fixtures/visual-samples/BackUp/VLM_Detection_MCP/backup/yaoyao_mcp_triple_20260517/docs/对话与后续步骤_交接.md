## 1. 项目目标（产品规格 v2）

## 8. 常见问题（本次对话已遇到）

| 现象 | 原因 | 处理 |
|------|------|------|
| `Could not resolve hostname company-linux` | 在 **Linux 服务器里** 用了 Windows 别名 | 在服务器上用 IP；别名只在 Windows/Cursor 连远程时用 |
| Cursor `Connection reset` | 网络/SSH 策略/安装远程组件中断 | 重连；config 加 `ServerAliveInterval 30`；查 `journalctl -u sshd` |
| 以为没有权重 | 搜索工具扫不到大 `.pt` | `ls -lh weights/*/*.pt` 确认 |
| Agent 无法自动 scp | 无 SSH 公钥，需交互密码 | 用户运行 `deploy_full_to_server.ps1` 或拖拽上传 |
